VirtualBox Setup Guide

  1. Download and install VirtualBox (www.virtualbox.org)
  2. In VirtualBox, click "new."
  3. Set the virtual machine type to "Microsoft Windows" and the version to "Windows 98."
  4. Click through the rest of the prompts. The default settings are fine.
  5. Open "settings" -> "storage."
  6. In the list of storage devices, select the disk icon labelled "empty" under "controller: IDE."
  7. Under "attributes," click the disk icon to open a dropdown menu and select "choose a disk file."
  8. Navigate to your Windows 98 SE .iso file.
  9. Click "OK."
  10. Start the virtual machine. When prompted to "select start-up disk," choose your .iso file.
  11. Windows can be set up normally from here. The Windows product key can be found in KEY.txt.

After Windows 98 SE is installed on your virtual machine, you may remove the .iso file from the machine using steps 5-9 (albeit without selecting the .iso file).

DOSBox-X Setup Guide

An excellent guide on how to set up Windows 98 SE on DOSBox-X can be found here. Note that a standard installation of DOSBox does not work - you must use another distribution. While the tutorial recommmends DOSBox Daum and JDOSBox, I recommend using only DOSBox-X for everything.

Internet Access Setup Guide

  1. From the desktop, open "start" -> "programs" -> "accessories" -> "internet tools" -> "internet connection wizard"
  2. Select "I want to set up my internet connection manually, or I want to connect through a local area network (LAN)."
  3. Select "I connect through a local area network (LAN)."
  4. Select "automatic discovery of proxy server (recommended)" (selected by default).
  5. Do not set up an internet mail account.
  6. Click through the rest of the prompts.

Note that you will only be able to connect to sites that do not enforce HTTPS. Very few modern sites will work.
